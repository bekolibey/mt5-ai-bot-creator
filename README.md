# mt5-ai-bot-creator

SQL History Demo - SQL sorgu geçmişi ve demo uygulaması.

## Kurulum

Projeyi klonlayın:
```bash
git clone https://github.com/bekolibey/mt5-ai-bot-creator.git
cd mt5-ai-bot-creator
```

## Kullanım

Projeyi tarayıcıda açmak için `index.html` dosyasını çalıştırın veya bir yerel sunucu başlatın:
```bash
npx serve .
```

## Teknolojiler

- HTML5
- CSS3
- JavaScript (Vanilla)
